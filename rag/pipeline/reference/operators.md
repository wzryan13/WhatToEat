## Milvus 过滤表达式速查

### 一、比较运算符（最常用）

用于数值或文本字段的精确比较：

* `==` 等于
* `!=` 不等于
* `>`  `<`  `>=`  `<=` 数值比较

示例：

```text
status == "active"
age >= 18
```

---

### 二、范围 / 模式匹配

#### `LIKE`（字符串匹配）

* `%` 表示任意长度字符
* **前缀匹配性能最好**

| 用途 | 示例                 |
| -- | ------------------ |
| 前缀 | `name LIKE "红烧%"`  |
| 后缀 | `name LIKE "%鸡"`   |
| 包含 | `name LIKE "%红烧%"` |

> ⚠️ 中缀 / 后缀匹配性能较差，谨慎使用

---

### 三、逻辑运算符

用于组合多个条件：

* `AND`：所有条件必须满足
* `OR`：满足任一条件
* `NOT`：条件取反

示例：

```text
category == "家常菜" AND difficulty == "简单"
dish_name LIKE "%鸡%" OR dish_name LIKE "%鱼%"
```

必要时使用括号：

```text
(category == "川菜" OR category == "湘菜") AND difficulty == "简单"
```

---

### 四、算术运算符（一般不推荐）

仅适用于数值字段，常见场景较少：

* `+  -  *  /  %  **`

示例：

```text
id % 2 == 0
```
